总课时

365天

标准学习时间90天

分享

第1课

下载并安装MonoDevelop

本节课我们主要学习下载安装Unity 和 MonoDevelop，并会创建并第一个 C# 工程运行开始学习(13'40")

第2课

C# 编译过程

本节课我们主要了解一下 C# 的编译过程，还会掌握语句的概念以及注释的用法开始学习(20'5")

第3课

二进制

本节课我们主要了解二进制的意义，以及二进制和十进制得相互转换，还会了解八进制、十六进制开始学习(20'44")

第4课

变量和常量

本节课我们主要学习变量的声明和初始化，以及常量的使用开始学习(22'3")

第5课

基本数据类型

本节课我们主要学习 C# 中常用的数据类型，包括整型、浮点型、字符串等类型开始学习(16'15")

第6课

运算符和表达式

本节课我们主要学习 C# 中的基本运算符，以及运算符的优先级和结合性，还会了解表达式的概念开始学习(24'11")

第7课

类型转换

本节课我们主要学习不同类型之间的相互转换，包括自动转换和强制转换开始学习(21'22")
